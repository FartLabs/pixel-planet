Pixel Planet provides a collection of interactive 3D pixel art planets. Like Shadcn, this is not a traditional library. You copy the components directly into your project to maintain full control over the implementation.

These components help you integrate high-quality, performant 3D assets without managing complex shader logic or 3D assets manually.

If you want to suggest new celestial bodies or report issues, open an issue on the [GitHub repository]({GITHUB_REPO_URL}).
