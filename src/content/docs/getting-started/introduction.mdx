`PixelPlanet` provides a collection of interactive 3D pixel art planets. Use these components to add celestial bodies to your React applications.

Like Shadcn, this is not a traditional library. You do not install it as a dependency. Instead, you copy the source code of a component into your project to maintain full control.

If you want to suggest new celestial bodies or report issues, open an issue on the [GitHub repository]({GITHUB_REPO_URL}).
