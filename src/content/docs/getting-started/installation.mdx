You can add Pixel Planet components to any React project using the `shadcn` CLI.

### Prerequisites

Before installing, ensure your project meets these requirements:

- **React**: A React-based project (e.g., Next.js, Vite).
- **React Three Fiber**: Install `@react-three/fiber` and `three` to render the 3D components.
- **Tailwind CSS**: The components use Tailwind for positioning and styling.

### Add components

To add a component, run the `shadcn add` command followed by the component identifier. This command downloads the source code and dependencies directly into your project.

<CliCommandCode
  action="run"
  command="shadcn@latest add pixel-planet/pixel-planet"
/>

If the short identifier fails, use the absolute URL to the component definition:

<CliCommandCode
  action="run"
  command={`shadcn@latest add ${SERVER_URL}/r/pixel-planet.json`}
/>
