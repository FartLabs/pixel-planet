If you have an idea for a new feature or celestial body, follow these steps to submit a request.

<Steps>

1. **Search existing issues**

   Check the [GitHub issues](https://github.com/FartLabs/pixel-planet/issues?q=is%3Aissue%20state%3Aopen%20label%3Aenhancement) to see if you are suggesting a known idea.

2. **Open an issue**

   [Create a new issue](https://github.com/FartLabs/pixel-planet/issues) using the **Feature request** template.

3. **Describe the feature**

   Explain the goal, expected behavior, and potential alternatives. Detailed requests help maintainers understand the requirement quickly.

</Steps>
