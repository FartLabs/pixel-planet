# Pixel Planet

A collection of accessible components built for use with Shadcn.

## Documentation

Visit https://fartlabs.github.io/pixel-planet to view the documentation.

## Contributing

Please read the [contributing guide](/CONTRIBUTING.md).

Before you commit, please ensure your code is properly formatted and linted:

```bash
npm run format
npm run lint
```
